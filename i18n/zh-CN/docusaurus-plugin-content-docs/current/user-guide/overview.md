## 与 Scratch 的主要区别

如果你来自 Scratch，以下是你在 Bilup 中会发现的主要增强功能：

### 性能改进
- **🚀 Turbo 模式**：项目运行速度提高 2-10 倍
- **📈 60+ FPS**：更高帧率的流畅动画
- **⚡ 优化编译器**：高级代码优化
- **🎮 更好的图形**：硬件加速渲染

### 增强的界面
- **🌙 深色模式**：多种深色主题可用
- **🎨 自定义主题**：创建和分享主题
- **📱 响应式设计**：适用于平板电脑和移动设备
- **🔧 可自定义布局**：根据你的偏好调整界面

### 强大的插件
- **80+ 个内置插件**：广泛的功能扩展
- **🎛️ 插件管理器**：轻松的安装和配置
- **🔌 扩展系统**：加载自定义扩展
- **🎨 视觉增强**：改进的编辑器外观

### 高级功能
- **📐 自定义舞台尺寸**：超出标准 480x360
- **🎯 精确控制**：逐帧动画控制
- **🔍 调试工具**：高级项目调试
- **📊 性能监控**：实时性能指标

## 快速开始

### 1. 第一步
1. **打开 Bilup** 访问 [https://editor.bilup.org/](https://editor.bilup.org/)
2. **尝试界面** - 一切都像 Scratch 一样工作!
3. **启用 Turbo 模式**(⚡)以查看性能提升
4. **在设置中切换到深色模式**以获得现代外观

### 2. 启用基本插件
这些插件将立即改善你的体验：

```
✓ 鼠标位置 - 查看光标坐标
✓ 积木计数 - 监控项目复杂性
✓ FPS 计数器 - 跟踪性能
✓ 变量管理器 - 更好的变量控制
✓ 查找栏 - 在脚本中搜索积木
```

### 3. 优化性能
配置这些设置以获得最佳性能：

```
高级设置：
✓ Turbo 模式：开
✓ 帧率：60 FPS
✓ 高质量画笔：开
✓ 插值：开
```

|--------|----------|
| 保存项目 | Ctrl+S(Cmd+S) |
| 加载项目 | Ctrl+O(Cmd+O) |
| 撤销 | Ctrl+Z(Cmd+Z) |
| 重做 | Ctrl+Shift+Z(Cmd+Shift+Z) |
| 运行项目 | Ctrl+Enter(Cmd+Enter) |
| 停止项目 | Ctrl+Shift+Enter(Cmd+Shift+Enter) |
| 全屏 | F11 |

## 常见工作流程

### 创建新项目
1. **从空白项目开始**或选择模板
2. **从角色库添加角色**或创建新角色
3. **通过从调色板拖动积木来创建脚本**
4. **使用绿旗频繁测试**
5. **定期保存**以防止数据丢失

### 优化项目性能
1. **启用 Turbo 模式**以加快执行
2. **将帧率提高到** 60 FPS 以提高流畅度
3. **使用高效脚本** - 避免不必要的循环
4. **用 FPS 计数器插件监控性能**
5. **清理未使用的资源**以减小文件大小

### 自定义你的体验
1. **选择适合你偏好的主题**
2. **为你的工作流程启用有用的插件**
3. **根据项目需要调整舞台尺寸**
4. **在设置中配置键盘快捷键**
5. **设置高效的工作区布局**
