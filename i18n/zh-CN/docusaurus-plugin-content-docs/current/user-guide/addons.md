# 插件

Bilup 包含一个强大的插件系统，允许你用自定义功能和增强功能扩展功能。

## 什么是插件?

插件是能够修改或扩展 Bilup 行为的 JavaScript 模块。它们可以：

- 添加新的 UI 元素
- 修改现有界面
- 添加新的积木或功能
- 自定义编辑器体验
- 与外部服务集成

## 内置插件

Bilup 附带了多个内置插件：

### 编辑器增强
- **积木计数**：显示项目中的积木数量
- **游戏手柄支持**：为项目启用游戏手柄输入
- **变量管理器**：高级变量管理工具
- **自定义积木形状**：自定义自定义积木的外观

### 开发工具
- **开发者工具**：高级调试和开发功能
- **性能监视器**：实时监控项目性能
- **控制台输出**：查看项目中的控制台消息

### 界面改进
- **深色模式**：在浅色和深色主题之间切换
- **紧凑模式**：为较小屏幕提供更紧凑的界面
- **自定义 CSS**：将自定义样式应用于编辑器

## 管理插件

### 启用/禁用插件

1. 点击右上角的 **设置** 按钮
2. 导航到 **插件** 选项卡
3. 使用开关切换插件的开启或关闭
4. 某些插件可能需要刷新页面才能生效

### 插件设置

许多插件都有可配置的设置：

1. 点击插件旁边的齿轮图标
2. 根据需要修改设置
3. 更改通常会立即应用

## 流行的社区插件

Bilup 社区创建了许多有用的插件：

- **项目注释**：为项目添加注释和文档
- **备份系统**：自动项目备份
- **导出工具**：额外的导出格式和选项
- **协作工具**：实时协作功能

## 插件故障排除

如果你在使用插件时遇到问题：

1. 尝试禁用有问题的插件
2. 检查插件是否与你的 Bilup 版本兼容
3. 清除浏览器缓存并重新加载
4. 向插件开发者报告问题

有关更详细的插件信息，请参阅 [插件内部结构](../gui-internals/addons/home) 文档。
