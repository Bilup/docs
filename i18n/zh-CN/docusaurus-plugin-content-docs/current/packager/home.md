# Bilup 打包器

:::info
在这里使用 Bilup 打包器：https://packager.bilup.org/
:::

Bilup 打包器将 Scratch 项目转换为 HTML 文件、ZIP 压缩包或用于 Windows、macOS 和 Linux 的可执行程序。它类似于 HTMLifier 和 forkphorus 打包器。

这是用于存放某些额外文档的位置。请使用左侧的侧边栏进行导航。
