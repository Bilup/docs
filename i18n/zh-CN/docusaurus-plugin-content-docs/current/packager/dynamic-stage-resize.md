# 动态调整舞台大小

:::info
此页面是关于 [Bilup 打包器](https://packager.bilup.org/) 的。
:::

动态舞台调整会更改舞台大小以匹配其显示的任何宽高比和分辨率。

舞台不是缩放的；[大小](/custom-stage-size) 实际上会改变。例如，如果用户在 1920x1080 显示器上启用全屏，舞台会调整为 1920x1080。如果用户将窗口调整为 1x1，它也会调整为该大小，因此你应该考虑添加最小尺寸检查。

几乎没有任何项目会正确处理此功能。要使你的项目兼容：

 - 首先，使其与 [自定义舞台尺寸](/custom-stage-size) 完全兼容(请参阅该页面了解信息)
 - 然后，将你的舞台尺寸检测逻辑更改为每帧运行一次，并确保始终更新项目的位置(是的，这效率低下且奇怪，但它足够快，目前是最佳选择)
