# Bilup 文档

[Bilup](https://editor.bilup.org/) 的技术文档 - 一个面向开发者的改良版 Scratch 环境。

## Bilup 是什么?

Bilup 是一个基于 TurboWarp 构建的 Scratch 修改版，专为开发者和高级用户设计：

- **增强型虚拟机** - 具有开发者功能的改良虚拟机
- **无云存储** - 仅本地执行
- **JavaScript 集成** - 增强的扩展功能
- **开发者工具** - 技术文档和 API
- **改良界面** - 删除默认角色和变量
