# 高质量画笔

高质量画笔允许画笔项目以更高分辨率渲染，而不是局限于 480x360，并在编辑器中禁用某些坐标舍入。

:::caution
高质量画笔会显著降低性能，尤其是在全屏模式下，因为需要绘制的像素要多得多。

并非所有项目在启用高质量画笔后都会看起来更好。
:::
