# 自定义舞台尺寸

将 Scratch 舞台的尺寸从 480x360（4:3）更改为其他尺寸。

:::caution
大多数项目在使用自定义舞台尺寸时无法正常工作。
:::

640x360 将创建一个宽屏 16:9 舞台。技术上可以使用更高分辨率如 1280x720、1920x1080 或更大，但我们建议使用相同宽高比的较低分辨率，并允许全屏模式将其放大到用户的显示器（请记住，某些显示器可能甚至无法容纳如此大的舞台！）。

项目可以使用诸如 https://scratch.mit.edu/projects/588493413/ 之类的方法检测舞台尺寸。
