# Scratch 账户与 Bilup

虽然你无法在 Bilup 上登录你的 Scratch 账户(除了 scratch.mit.edu 之外的任何要求你提供 Scratch 密码的网站都是诈骗——没有例外)，你仍然可以在不分享的情况下运行项目、上传项目到 Scratch，以及使用你自己的用户名。

**要在不分享的情况下运行用 Scratch 制作的 Bilup 项目**：在 Scratch 编辑器中使用「文件 > 保存到电脑」将项目保存到电脑上的文件，然后在 Bilup 中选择「文件 > 从电脑加载」来加载该文件。

**要上传在 Bilup 中制作的项目到 Scratch**：在 Bilup 的编辑器中使用「文件 > 保存到电脑」将项目保存到电脑上的文件，然后在 Scratch 的编辑器中使用「文件 > 从电脑加载」来选择该文件。

**要更改 Bilup 中的用户名**：使用「编辑 > 更改用户名」。

即使你在另一个标签页中登录了 Scratch，Bilup 也无法访问它，因为浏览器阻止网站能够访问其他网站的登录凭据(这正是保证网上银行安全的原因)。我们必须要求你提供 Scratch 密码，这将带来无法修复的安全隐患，而且可能会被 Scratch 屏蔽。

根据你想要做的事情，你可能会发现使用 [Bilup 打包器](https://packager.bilup.org/) 能够将项目上传到通用 HTML 或游戏托管网站。
