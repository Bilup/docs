# renderer.getBoundsForBubble()

获取与可绘制对象关联的文本气泡的边界框。

## 语法

```javascript
renderer.getBoundsForBubble(drawableID)
```

## 参数

### drawableID
**类型:** `number`

拥有气泡的可绘制对象的 ID。

## 返回值

**类型:** `object`

边界框对象（left, right, top, bottom, width, height）。
