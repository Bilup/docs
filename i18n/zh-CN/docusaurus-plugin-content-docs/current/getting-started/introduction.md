# Bilup 简介

Bilup 是一个功能强大、内容丰富的 Scratch 修改版，它在原始平台的基础上增强了性能优化、扩展了自定义选项，并提供了全面的插件系统。

## Bilup 的特别之处

### 🚀 性能增强
- **Turbo 编译**：项目在我们优化的编译器中运行得更快
- **高级渲染**：改进的图形性能和减少的延迟
- **内存优化**：更好地管理大型项目的内存
- **60+ FPS 支持**：更高帧率的流畅动画

### 🎨 主题与自定义
- **深色主题**：多种深色主题变体，保护眼睛
- **自定义主题**：创建和分享自己的主题
- **界面自定义**：自定义编辑器的布局和外观
- **CSS 变量**：微调视觉设计的每个细节

### 🔧 插件系统
- **80+ 内置插件**：通过我们丰富的插件库增强功能
- **自定义插件**：使用我们强大的 API 开发你自己的插件
- **插件管理器**：轻松安装和管理插件
- **实时配置**：无需重新加载即可更改插件设置

### 💻 开发者工具
- **高级调试器**：单步执行代码、设置断点、检查变量
- **性能分析器**：分析项目性能并优化瓶颈
- **扩展开发**：创建自定义积木和扩展
- **API 访问**：对编辑器和运行时进行编程控制

## 与 Scratch 的主要区别

| 功能 | Scratch | TurboWarp | Bilup |
|---------|---------|----------|----------|
| **性能** | 慢 | 快 | 高度优化的编译器 |
| **主题** | 仅浅色 | 深色模式 | 自定义主题 |
| **插件** | 无 | 80+ 个内置插件 | ~90 个内置插件 |
| **帧率** | 30 FPS | 60 FPS | 自定义 FPS |
| **自定义** | 有限 | 有限 | 高度可自定义 |

## 架构概述

Bilup 构建在模块化架构上，由几个关键组件组成：

- **scratch-gui**：用户界面和编辑器
- **scratch-vm**：执行项目的运行时引擎
- **scratch-render**：角色和舞台的渲染引擎
- **scratch-blocks**：可视化积木编辑器
- **插件系统**：模块化增强系统
- **主题引擎**：动态主题和样式系统
