# 快速入门指南

几分钟内开始使用 Bilup! 本指南将帮助你创建第一个项目并了解与 Scratch 的主要区别。

## 创建你的第一个项目

1. **打开 Bilup** - 访问 [editor.bilup.org](https://editor.bilup.org/)
2. **熟悉界面** - 界面与 Scratch 类似，但有一些增强功能
3. **开始创作** - 像在 Scratch 中一样开始添加角色和积木

## 与 Scratch 的主要区别

### 🚀 增强的性能
- 优化的编译器，使项目运行得更快
- 支持更高的帧率(最高 60+ FPS)
- 改进的渲染，使动画更流畅

### 🎨 自定义选项
- **深色主题** - 在浅色和深色主题之间切换
- **自定义主题** - 创建和应用你自己的主题
- **舞台尺寸** - 自定义舞台尺寸，超出标准的 480x360

### 🔧 开发者功能
- **浏览器控制台访问** - 使用 `window.vm` 访问虚拟机
- **高级扩展** - 使用 JavaScript 创建自定义积木
- **性能分析** - 监控和优化项目性能

### 💻 默认角色
- Bilup 删除默认角色和变量。
- 所有 Scratch 角色仍然可以在角色库中使用

## 你的第一个 Bilup 项目

让我们创建一个简单的项目来熟悉 Bilup：

### 第一步：添加角色
1. 点击右下角的 **选择一个角色** 按钮
2. 从角色库中选择任意角色
3. 角色出现在舞台上

### 第二步：添加一些代码
1. 从「事件」类别中拖动 **当绿旗被点击** 积木
2. 从「外观」类别中添加 **说「你好! 」2 秒** 积木
3. 从「运动」类别中添加 **移动 10 步** 积木
4. 添加一个 **重复执行** 积木使其重复

### 第三步：运行你的项目
1. 点击舞台上方的 **绿旗**
2. 观看你的角色移动和说话! 

## 探索高级功能

### 使用编译器
Bilup 自动将你的积木编译成 JavaScript 以加快执行速度。你可以：
- 在浏览器控制台中查看编译后的代码
- 分析性能以发现瓶颈
- 根据分析结果优化你的项目

### 自定义扩展
使用 JavaScript 创建自定义积木：
1. 打开 **扩展** 菜单
2. 浏览可用的扩展
3. 从 URL 加载自定义扩展
4. 访问标准 Scratch 中不可用的高级 API

### 保存和分享

**本地保存：**
- 文件 → 保存到电脑
- 项目保存为与 Scratch 兼容的 `.sb3` 文件

**保存到云端：**
- 创建账户以在线保存项目
- 从任何设备访问你的项目
- 与他人分享项目链接

## 常见任务

### 导入 Scratch 项目
1. 从 Scratch 下载你的 `.sb3` 文件
2. 在 Bilup 中：文件 → 从电脑加载
3. 选择 `.sb3` 文件
4. 你的项目将加载，所有角色和代码都完好无损

### 更改主题
1. 点击菜单栏中的 **外观** 图标
2. 从下拉菜单中选择 **主题**
3. 从可用主题中选择或创建你自己的主题

### 调整性能
1. 从菜单中访问 **高级设置**
2. 调整帧率(30 FPS 或 60 FPS)
3. 启用/禁用编译器以进行兼容性测试
4. 配置舞台尺寸和渲染选项

## 键盘快捷键

- **Ctrl/Cmd + S** - 保存项目
- **Ctrl/Cmd + Shift + S** - 另存为新文件
- **Ctrl/Cmd + O** - 打开项目
- **Ctrl/Cmd + Enter** - 启动/停止项目(绿旗/停止按钮的替代方式)
- **Ctrl/Cmd + Shift + E** - 导出项目
