# 从 Scratch 迁移

Bilup 可以运行大多数 Scratch 项目，并提供增强的性能和附加功能。

## 项目兼容性

### 支持的功能
- 所有标准 Scratch 积木
- 自定义积木和过程
- 大多数扩展
- 角色、造型和声音

### 增强的功能
- 更快的执行速度
- 额外的扩展 API
- 自定义舞台尺寸
- 开发者工具集成

## 加载 Scratch 项目

1. 从 Scratch 网站下载 `.sb3` 文件
2. 在 [editor.bilup.org](https://editor.bilup.org/) 打开 Bilup
3. 文件 → 从电脑加载
4. 选择 `.sb3` 文件

## 主要区别

### 技术变化
- 删除默认角色和变量。
- 无云变量（仅本地执行）
- 增强的 JavaScript 扩展功能
- 为更好性能而改良的虚拟机

### 开发者功能
- 用于调试的浏览器控制台访问
- 扩展开发 API
- 自定义积木创建
- 性能监控工具

## 故障排除

### 常见问题
- **项目无法加载**：检查文件格式（必须是 `.sb3`）
- **积木缺失**：某些自定义扩展可能不可用
- **性能差异**：Turbo 模式可能会改变计时行为

### 解决方案
- 使用「从 Scratch 网站加载」来解决兼容性问题
- 检查浏览器控制台中的错误消息
- 如果计时至关重要，禁用 Turbo 模式
