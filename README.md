## Local Development

This website is built with [Docusaurus](https://docusaurus.io/). Documentation is inside the `docs` folder, generally that is all you will have to touch. ~~We accept pull requests.~~ We don't accept pull requests for now, but may be open in the future.

```bash
npm install
# For development:
npm start
# For production:
npm run build
```
